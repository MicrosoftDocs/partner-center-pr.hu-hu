## <a name="now-available-windows-7-year-2-esu-for-cloud-solution-provider-csp"></a><a name="6"></a>Már elérhető a Windows 7 év 2 EUME a Cloud Solution Provider (CSP) számára

### <a name="categories"></a>Kategóriák

- Dátum: 2020-06-29
- Típus: ajánlatok/piacok
- Állapot: már elérhető

### <a name="summary"></a>Összefoglalás

A Windows 7, 2. év kiterjesztett biztonsági frissítések (EUME) díjszabása mostantól [az erőforrás-gyűjteményben](https://partner.microsoft.com/resources/collection/windows-esu-csp#/)található.

### <a name="impacted-audience"></a>Érintett közönség

A Cloud Solution Provider (CSP) programban résztvevő összes partner

### <a name="details"></a>Részletek

A Windows 7 támogatása 2020 január 14-én fejeződött be. Míg számos ügyfél jól használható a Windows 10 központi telepítésére, a Microsoft tudomásul veszi, hogy mindenki a frissítési folyamat egy másik pontján van. A Windows 7 Year 1 EUME érvényességi időszaka 2021 január 12-én végződik. Tudjuk, hogy ügyfeleink megtervezik a következő évet.  

Azon ügyfelek számára, akik továbbra is a Windows 7 rendszerű eszközöket használják a 2021-es verzióban, a Windows 7 év 2 EUME-es verziójának díjszabását a Cloud Solution Provider (CSP) partnerei számára elérhetővé tesszük, hogy a Windows 10-es döntéssel segítse ügyfeleit. További információért látogasson el [az erőforrás-galériába](https://partner.microsoft.com/resources/collection/windows-esu-csp#/) .

### <a name="next-steps"></a>További lépések

## <a name="partner-center-relationship-removal-feature-update"></a><a name="5"></a>Partner Center-kapcsolat eltávolítási funkciójának frissítése

### <a name="categories"></a>Kategóriák

- Dátum: 2020-06-29
- Típus: képességek
- Állapot: már elérhető
- 
### <a name="summary"></a>Összefoglalás

2020. június 29-én a Cloud Solution Provider (CSP) közvetett szolgáltatók és a közvetett viszonteladók is eltávolíthatják az alárendelt kapcsolatokat a fiókpartner használatával, és többé nem kell elküldeni a partneri központ támogatási jegyét. Így jelentős mennyiségű időt takaríthat meg a partnerek számára ezen tevékenységek végrehajtásakor.

### <a name="impacted-audience"></a>Érintett közönség

Cloud Solution Provider (CSP) közvetett szolgáltatók és közvetett viszonteladók

### <a name="scenarios-supported"></a>Támogatott forgatókönyvek:

- A közvetett viszonteladóval létesített kapcsolattal rendelkező közvetett szolgáltatók el tudják távolítani ezt a kapcsolatot a partner Center használatával.
- A végfelhasználók által létesített kapcsolattal rendelkező közvetett viszonteladók eltávolítják ezt a kapcsolatot a partner Center használatával.

A fenti forgatókönyvekben szereplő műveleteket egy partner Center felügyeleti ügynöknek kell végrehajtania.

### <a name="next-steps"></a>Következő lépések:

- További információért látogasson el [az erőforrás](https://partner.microsoft.com/resources/collection/partner-center-relationship-removal#) -katalógusba.
- Tekintse át az Operations Readiness Gallery-gyűjtemény anyagait: a fiókpartner-kapcsolat eltávolítási funkciójának frissítése című témakört.
- Ossza meg ezt az információt a szervezeten belül, hogy a megfelelő csapat megtekintse és frissítse a folyamatokat.

## <a name="billing-frequency-default-changing-from-annual-to-monthly"></a><a name="4"></a>A számlázási gyakoriság alapértelmezett változása az évesről a hónapra

### <a name="categories"></a>Kategóriák

- Dátum: 2020-06-17
- Típus: képességek
- Állapot: hamarosan elérhető

### <a name="summary"></a>Összefoglalás

A számlázási gyakoriság alapértelmezett értékének változása a partner Center vásárlási élményének éves és havi időpontjában történik.

### <a name="impacted-audience"></a>Érintett közönség

Cloud Solution Provider (CSP) közvetlen szolgáltatók és közvetett szolgáltatók

### <a name="details"></a>Részletek

A számlázási gyakoriság beállítása határozza meg, hogy a partner milyen gyakran vásárolja meg a megvásárolt szolgáltatásokat. A partnerek ezt a beállítást a **felülvizsgálati** lépésben tekintik meg a partner Center katalógusának megvásárlásakor. Ez a beállítás alapértelmezés szerint **éves** , mert a licenc-alapú szolgáltatások esetében az éves számlázás bemutatkozott. Partneri visszajelzések alapján a partner Center csapata az **alapértéket** június **17-én** megváltoztatja, így jobban igazodik a partneri elvárásokhoz. A partnerek ezt megváltoztathatják, mielőtt elküldenek egy tranzakciót a felülvizsgálati oldalon vagy egy tranzakció után az előfizetés kezelése lapon. Vegye figyelembe, hogy a csak az éves számlázást támogató ajánlatok továbbra is az alapértékek szerint jelennek meg. Ez az alapértelmezett módosítás csak a partneri központ portál felhasználói felületére vonatkozik, a partner Center API-k működésének változása nem változik.

Reméljük, hogy ez a változás csökkenti annak a lehetőségét, hogy a partner véletlenül vásárolja meg az évente számlázott előfizetéseket.

## <a name="changes-to-microsoft-365-f1"></a><a name="3"></a>Microsoft 365 F1 változásai

### <a name="categories"></a>Kategóriák

- Dátum: 2020-06-15
- Típus: ajánlatok/piacok
- Állapot: már elérhető

### <a name="summary"></a>Összefoglalás

A Microsoft 365 F1 új szolgáltatásokkal lesz frissítve. A rendszer új ajánlatot tesz közzé, hogy az előző F1-et június 15-én cserélje le.

### <a name="impacted-audience"></a>Érintett közönség

Cloud Solution Provider (CSP) közvetlen szolgáltatók és közvetett szolgáltatók

### <a name="details"></a>Részletek

A Microsoft 365 F1 a partner Centerben lett elérhető a 2020. április óta. Az új szolgáltatások támogatása az Exchange kioszk és az MS Search funkcióinak hozzáadásával történik. Az új szolgáltatásokat a rendszer automatikusan kiépíti a meglévő előfizetéseken az eredeti F1-es ajánlatok alapján június végéig. Az eredeti F1-es ajánlatokat június 15-én, az új előfizetés-vásárlások esetén pedig az új F1-es ajánlatokkal (alább) cseréljük.

Eredeti F1-ajánlatok:

   |**Ajánlat neve**|**Ajánlat azonosítója**|**Anyag azonosítója**|**Kiépítési azonosító**|
   |-------------------|:------|:------|:------|
   |Microsoft 365 F1|1ce4a4ba-65f7-49e7-bb28-d92d07d58b86|1PI – 00005|M365_F1|
   |Microsoft 365 F1 próbaverzió|dd97e36a-f0df-4a9a-b461-625420656511|1PI – 00006|M365_F1|

Az új F1-es előfizetéseket megvásárolni kívánó partnereknek a következő ajánlat-azonosítókat kell használniuk június 15-én. A partnerek láthatják ezeket az azonosítókat a partner Center-katalógusban. A partner Center API-kat használó partnereink az alábbi információkat is használhatják.

Új F1-ajánlatok:

   |**Ajánlat neve**|**Ajánlat azonosítója**|**Anyag azonosítója**|**Kiépítési azonosító**|
   |-------------------|:------|:------|:------|
   |Microsoft 365 F1|17b4a1d6-2c65-478e-a046-51cf40d25d76|1PI-00012|M365_F1_COM|
   |Microsoft 365 F1 próbaverzió|2e4e9424-06db-4d35-b2d8-3a1b0ef1dc63|1PI-00013|M365_F1_COMM|

### <a name="call-to-action"></a>Művelet meghívása
A jelenlegi Microsoft 365 F1 termékkel rendelkező partnerek nem rendelkeznek művelettel. Az Exchange kioszk és az MS Search szolgáltatási csomagjait a meglévő visszatöltötte június végéig fogja megadni.

Azok a partnerek, akik június 15-én szeretnék megvásárolni Microsoft 365 F1-et, az új F1-es információk használatával biztosítják a megfelelő ajánlat megvásárlását. Az új F1-es ajánlatok AZONOSÍTÓját és egyéb információit a júliusi Cloud Solution Provider (CSP) árlista frissítése július 1-jén frissíti a rendszer.
